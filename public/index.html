How To Hai·koo:

  • Scroll down and enter a haiku in the section below, one line in each input area.
  • A haiku is a poem, structured as follows:
    5 syllables for the first line,
    7 syllables for the second,
    and 5 syllables for the last.
  • Subsequent inputs will activate once the previous line's syllables have been approved.
  • This API is picky. Stick to single-syllabled English root words (run vs. running, bus vs. busses).
  • The API isn't perfect, it thinks 'banana' is two syllables, and 'achieve' is one.
  • Keep scrolling to see other haikus, and vote!
